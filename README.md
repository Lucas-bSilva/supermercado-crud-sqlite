# Supermercado — CRUD de Produtos (Parte 1) com SQLite (v3)

Projeto focado apenas em Produtos (Estoque), com o menu exatamente nas opções exigidas:

1. Inserir  
2. Alterar  
3. Pesquisar por nome  
4. Remover  
5. Listar todos  
6. Exibir um  

Extras coerentes com a especificação:
7. Relatório de estoque (resumo)  
9. Resetar banco (apagar supermercado.db)  
0. Sair  

## Como rodar
```bash
python main.py
```
O banco `supermercado.db` será criado automaticamente.
